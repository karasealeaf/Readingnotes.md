## What is Markdown?
Created in 2004 by John Gruber, Markdown is used to add formatting element to a plaintext document, it is essentially a lightweight markup language.

Here is a list of some of the meanings:
-	#Heading one
- ##Heading two
- ###Heading three etc.
-	**Bold**
